Walls

Consider N walls, each of unit width, situated next to each other.
You have to select any two walls i and j (1i,jN) such that if you break all the walls except i and j and fill the spaces between them with water, then the amount of water stored is maximum.

Write a program to find the maximum amount of water that can be accumulated between the walls.

Input format

  • First line: T (number of test cases)
    For each test case
    • First line: N
    • Second line: N space-separated integers (denoting the heights of the walls)

Output format

For each test case, print the maximum amount of water that can be stored between the walls.

Constraints

1T100
1N104
1Value109

Sample Input
2

5

1 2 3 4 5

5

3 2 1 4 5
Sample Output
4

9
Explanation

For 1st testcase if we choose wall 2 and wall 5 . The amount of water that can be stored will be distance between walls i.e (5-2-1)=2. Hence answer is 2* min(height[2],height[5]) = 4 which is maximum.

For 2nd testcase if we choose wall 1 and wall 5 . The amount of water that can be stored will be distance between walls i.e (5-1-1)=3. Hence answer is 3* min(height[1],height[5]) = 9 which is maximum.
